## Overview
Exploratory Data Analysis (EDA) of an online retail transactional dataset to understand **customer purchasing behavior, product demand, and sales trends** using Python.

---

## Tools
Python • Pandas • NumPy • Matplotlib • Seaborn • Jupyter Notebook

---

## Key Analysis
- Data cleaning and preprocessing
- Sales distribution analysis
- Top-selling products
- Country-wise sales patterns
- Time-based purchase trends

---

## Outcome
Extracted insights about **customer behavior and product demand** through statistical analysis and data visualization.
