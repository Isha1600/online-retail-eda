# Online Retail Exploratory Data Analysis

## Project Overview
This project analyzes transactional data from an online retail store to discover purchasing patterns and customer behavior.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Analysis
- Sales distribution
- Top selling products
- Country wise purchases
- Customer behavior

## Dataset
Online Retail Dataset
